# To-Do Web Application

## 📌 Task
SkillCraft Technology Web Development Internship – Task 4

## 📖 Project Overview
The To-Do Web Application is a simple and responsive task management application that helps users organize their daily activities. Users can add new tasks, mark tasks as completed, and delete tasks easily.

## 🚀 Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Clean and responsive user interface
- Easy to use

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript
